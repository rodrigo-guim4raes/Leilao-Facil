# LeilãoFácil

## Sobre o projeto
Como parte dos meus estudos do curso técnico de Análise e Desenvolvimento de Sistemas no Senac, desenvolvi um sistema desktop em Java para o gerenciamento de uma casa de leilões. 
O **LeilãoFácil** é um sistema informatizado para apoiar as rotinas de uma casa de leilões, organizando informações como clientes, itens/lotes e lances.  
O objetivo é centralizar dados e agilizar processos, trazendo mais controle e rastreabilidade para as operações.

## Tecnologias utilizadas
- Java
- MySQL
- Git / GitHub
